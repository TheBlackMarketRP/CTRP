## How to Install ##
1. Add to server.cfg 'ensure ull-essential'
2. Adjust the settings in the configuration files as you wish, and restart the server.
3. Leave the script folder name as 'ull-essential' for all functions and UI to work properly.
4. Do not restart the script while the server is running, otherwise some functions will not work properly.
